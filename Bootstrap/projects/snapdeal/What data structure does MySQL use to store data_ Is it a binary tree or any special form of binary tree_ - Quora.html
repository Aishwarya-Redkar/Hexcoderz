Answered Dec 17, 2012
The full answer is much too detailed to post here, but the brief answer is yes, the most common storage format in MySQL is InnoDB, which stores all its data and indexes as a type of B+ Tree data structure.

See also: B+ tree on wikipedia.

For more details, I'll refer you to the MySQL Internals Manual or the MySQL manual on Optimization and Indexes.
